## Installation

Before installing the application please make sure to have installed and in your `$PATH`:

* Node >= 8.9.4
* Yarn >= 1.3.2

Then run in a console:

```sh
yarn install
```

## Run a local instance

In a console run:

```sh
yarn start
```

Then open your browser on http://localhost:3000.

## Build the app for production

In a console run:

```sh
yarn build
```
